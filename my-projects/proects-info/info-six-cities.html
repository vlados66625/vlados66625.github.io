Six cities

В проекте Six-cities я разработал приложение на React с использованием TypeScript. В качестве сборщика настроен Vite, так как это современный и быстрый инструмент. В приложении настроен роутинг с помощью React Router, включая приватные маршруты для аутентифицированных и неаутентифицированных пользователей. Для управления состоянием используется Redux, а для асинхронных операций с сервером — Redux-Thunk. Запросы к серверу выполняются с помощью Axios. Для оптимизации работы компонентов применены хуки (useMemo, useCallback, memo), а также селекторы с createSelector. Код организован по принципу разделения на переиспользуемые компоненты, что упрощает поддержку и масштабируемость проекта. Для обеспечения качества кода написаны юнит- и интеграционные тесты с использованием Vitest.
