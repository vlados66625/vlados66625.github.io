Фермерские продукты

Фермерские продукты" — веб-приложение, разработанное на React. В проекте настроен роутинг, а для интерактивности используются хуки. Стилизация выполнена по принципу CSS-in-JS с использованием styled-components. В качестве сборщика используется CRA с внесенными изменениями под нужды проекта. Глобальное состояние управляется с помощью Context API. Для организации компонентов пользовательского интерфейса используется инструмент Storybook.
